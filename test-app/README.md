# Test Vue Application

Simple Vue 3 + Vite app for testing vite-browser.

## Setup

```bash
pnpm install
pnpm dev
```

Then in another terminal:

```bash
cd ..
pnpm dev open http://localhost:5173
pnpm dev detect
pnpm dev vue tree
```

# 版本说明

这个页面提供中文版的版本线说明，详细变更仍以英文 release notes 原文为准。

## 当前版本线

### v0.3.3

`v0.3.3` 是 `v0.3` 版本线上的稳定性补丁，重点在于提升真实 Vite 7 / Vue / Pinia 热更新故障里的证据拼接质量。

主要价值：

- 当当前错误栈落在下游组件时，错误关联仍更容易保留正确的源模块方向
- store 侧和 render 侧证据到达不均匀时，`changedKeys` 和 source module 提示更稳定
- 页面恢复后会清掉过期 runtime error，避免旧错误一直污染当前判断

## 怎么读这些版本

- `v0.2.x` 可以看作从“观察”走向“诊断”的开始
- `v0.3.x` 可以看作传播线索和实时复现可靠性持续增强的一条版本线
- patch 版本的重点通常不是改产品模型，而是让真实复现中的证据链更稳定、更可信

# 版本说明

## 当前版本

### v0.3.3

`v0.3.3` 是 `v0.3` 版本线上的稳定性补丁，重点提升了真实 Vite / Vue / Pinia 热更新故障中的证据拼接质量。

主要改进：

- 当错误栈落在下游组件时，错误关联仍能保持正确的源模块方向
- store 侧和 render 侧证据不均匀到达时，`changedKeys` 和源模块提示更稳定
- 页面恢复后清除过期的 runtime error，避免旧错误持续干扰当前判断

## 版本线

| 版本 | 定位 |
|---|---|
| `v0.2.x` | 从"观察"走向"诊断" —— 引入错误关联和规则化诊断 |
| `v0.3.x` | 传播线索和实时复现可靠性持续增强 |

Patch 版本的重点通常不是改产品模型，而是让真实复现中的证据链更稳定、更可信。
